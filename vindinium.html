Vindinium

What is Vindinium?

Vindinium is an artificial intelligence programming game where the players need to create their own code to participate. The code is written to create a bot whose objective is to collect the most gold by the end of the game. The bots can collect gold by collecting mines or stealing mines from other bots. Through the code, you can design your bot's strategy (e.g. attacking other bots, finding weak bots, etc.).

Global Data

This part of the Global Data code holds the position of my bot using an array.

This part of the Global Data code first determines which bots are the enemies and then pushes them into the array called enemyBots. This is necessary so that my bot is not mistaken for an enemy bot. The second array determines the positions of the enemy mines.
