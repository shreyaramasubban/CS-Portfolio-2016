How to Make the Hardware Page

Navbar

First I made an unordered list of links for each of the pages of the navbar, and I set the id as “navbar.” I opened a CSS file for the navbar and set the background color for the navbar to be “aliceblue” and the border for the navbar to be dashed purple and have a thickness of 2px. I used the margin command to set the space between each link and madea pink solid border around each one. Then, I made individual html files for each of the pages on the navbar (e.g. Hardware, Todo List), and I connected all of them to the navbar using “href.”

This command created the css file for the navbar.

I used the <li> tag to make the links for each of the navbar pages. <ul> formed an unordered list which id connected to the css file for the navbar. The <a> tag linked each page and marked the visited and unvisited pages with blue and purple.

I used the css file to style the navbar. Using “#navbar,” linked the css file to the unordered list in the html file. While styling, I used padding to determine how much space (px) I wanted between each line; I used margin to determine how much indentation I wanted, and I set the display to inline-block so that all the links from the navbar showed up in one line.

Headers

For the title, I used the <h1>, <center>, and <font> tags. Inside the <h1> tag I could style the text by setting the color of the text (indigo) and the font that I wanted to use (Georgia). For the subheader, I used the <h2>, <center>, <font>, and <u> tags. Inside the <h2> tag, I set the style, and I used the <u> tag to underline the text.

For the title of the page, I used <h1>, and I set the font to size 6 and the color to indigo. The second header is similar except, I used <h2> and I underlined the text, set the color to crimson, and the font size to 5.

Paragraphs and Images

For the paragraphs, I used the <p> and <font> tags. For the images, I used the <img> tag and inside of it I put the link for the image. I used the <div> tag to style the picture so that it was aligned in the center.

For the title of the page, I used <h1>, and I set the font to size 6 and the color to indigo. The second header is similar except, I used <h2> and I underlined the text, set the color to crimson, and the font size to 5.
