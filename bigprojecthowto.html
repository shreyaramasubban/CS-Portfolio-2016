How to Make the Trivia Page

First I linked the trivia page to the Navbar and added the background image to the page.

Creating the Questions

In the html, I created and formatted the questions for the trivia. For the input type, I used the radio button so that I could design the questions in a multiple choice format. I also used the onclick event to store choices a, b, c, or d, depending on which choice the user clicked on. I repeated this for all the questions.

Result:

I created a "Show answers" button for the user to click when the quiz is completed using the button tag and id.

Defining JavaScript Variables

In the beginning of my JavaScript, I created variables and set them to different variables from the html using "document.getElementById." I connected the button to my JavaScript and I created variables for the each of the answers that the user chose in the quiz.

Functions

I wanted the program to run when I clicked the show answers button, so, I created a function for when the button is clicked using ".addEventListener" and "('click', function()." In the function, I set if statements so that if the ans variable equals the right answer the message would be "Well Done!" and if ans did not equal the right answer, a "try again" message would appear. To make the messages pop up, I used the alert method.

Alert:

I also created a function to help determine whether the answer the user chose was the right answer. The function stores the answer chosen as the ans variable.
