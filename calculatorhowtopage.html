How to Make the Calculator Page

First I linked the calculator page to the Navbar and added the background image to the page.

Defining HTML Variables

In the html, I defined all the variables that I would use in the javascript. I used the <input> tag and id command to define the input variable as a number. I then created a drop-down list with 6 options using the <option> tag; I set the "id" of operation variable to the option that the user chose from the list. I created a variable for the "calculate" button using the <button> tag that operates the calculator. Also, I created a variable to display the answer.

Defining JavaScript Variables

In the beginning of my JavaScript, I created variables and set them to different variables from the html using "document.getElementById." I basically connected the equivalent variables in my html and JavaScript.

Functions

For my calculator, I wanted the program to run when I clicked the "calculate" button. So, I created a function for when the button is clicked using ".addEventListener" and "('click', function()." In the function, I first set the two input number variables and the operation variable to the value of those using the ".value" command. I then created an if then statement for each of the 6 operations that I wanted my calculator to do. For each if then statement, I used "alert" to display my answer.

For subtraction, I set the answer equal to the difference of inputnum1 and inputnum2.
